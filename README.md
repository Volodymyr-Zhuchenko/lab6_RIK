# lab6_RIK

Веб-застосунок «PromptVault» — менеджер AI-промптів, розроблений для лабораторної роботи №6 з курсу «Розробка інтерфейсів користувача».

## Опис проєкту

PromptVault — це SPA для зберігання, організації та пошуку AI-промптів. Застосунок дозволяє створювати, редагувати та видаляти промпти, позначати їх як обрані, фільтрувати за тегами та категоріями, а також миттєво копіювати вміст у буфер обміну. Всі дані зберігаються у `localStorage` браузера без потреби у бекенді; інтерфейс повністю україномовний, виконаний у темній темі з фіолетовими акцентами.

## Технологічний стек

* **Мова програмування:** JavaScript (ES2022, ES Modules)
* **UI-фреймворк:** Vue 3 (Composition API, `<script setup>`)
* **Маршрутизація:** Vue Router 4 (hash-mode)
* **Збірник:** Vite 5
* **Стилізація:** Tailwind CSS v3 + PostCSS + Autoprefixer
* **Збереження даних:** localStorage (ключ `ai-prompts-uk`)

## Функціональність

| Функція | Опис |
|---|---|
| Галерея промптів | Адаптивна сітка (1 → 2 → 3 колонки) з картками всіх промптів |
| Пошук | Повнотекстовий пошук за назвою, вмістом та тегами в реальному часі |
| Фільтрація за тегами | Кліком на тег фільтрується список; повторний клік скасовує фільтр |
| Категорії | 5 категорій: Coding, Writing, Analysis, Image, Other — з кольоровими бейджами |
| Обрані (Favorites) | Позначення промптів зіркою; окрема сторінка з лічильником у навбарі |
| Копіювання | Кнопка миттєвого копіювання тексту промпту у буфер обміну |
| Створення / Редагування | Форма з валідацією: назва, текст, теги, категорія |
| Видалення | Видалення з підтвердженням |
| Toast-сповіщення | Анімовані повідомлення про успішні та помилкові дії |
| Seed-дані | 7 демонстраційних промптів завантажуються при першому відкритті |

## Структура проєкту

```text
laba/
├── index.html                  # HTML-шаблон Vite
├── vite.config.js              # Конфігурація збірника
├── tailwind.config.js          # Конфігурація Tailwind CSS
├── postcss.config.js           # Конфігурація PostCSS
├── package.json                # Залежності та скрипти
└── src/
    ├── main.js                 # Точка входу Vue
    ├── App.vue                 # Кореневий компонент (navbar + router-view)
    ├── assets/
    │   └── main.css            # Tailwind-директиви + глобальні стилі
    ├── components/
    │   └── PromptCard.vue      # Картка промпту (копіювання, обране, видалення)
    ├── composables/
    │   └── usePrompts.js       # Вся бізнес-логіка: CRUD, пошук, теги, обрані
    ├── router/
    │   └── index.js            # Маршрути застосунку
    └── views/
        ├── HomeView.vue        # Головна: галерея + пошук + фільтр тегів
        ├── AddPromptView.vue   # Форма створення промпту
        ├── EditPromptView.vue  # Форма редагування промпту
        ├── FavoritesView.vue   # Сторінка обраних промптів
        └── NotFoundView.vue    # 404
```

## Маршрути

| Шлях | Компонент | Опис |
|---|---|---|
| `/` | HomeView | Перегляд усіх промптів з пошуком і фільтрами |
| `/add` | AddPromptView | Створення нового промпту |
| `/edit/:id` | EditPromptView | Редагування промпту за ID |
| `/favorites` | FavoritesView | Обрані промпти |
| `/*` | NotFoundView | Сторінка 404 |

## Composable `usePrompts()`

| Що | Тип | Опис |
|---|---|---|
| `prompts` | `ref` | Масив усіх промптів |
| `searchQuery` | `ref` | Поточний пошуковий запит |
| `activeTag` | `ref` | Обраний тег-фільтр |
| `filteredPrompts` | `computed` | Промпти після пошуку та фільтрації |
| `favorites` | `computed` | Масив промптів з `isFavorite: true` |
| `allTags` | `computed` | Унікальні теги з усіх промптів |
| `addPrompt(payload)` | метод | Додати новий промпт |
| `updatePrompt(id, payload)` | метод | Оновити промпт |
| `deletePrompt(id)` | метод | Видалити промпт |
| `toggleFavorite(id)` | метод | Додати / зняти з обраних |
| `setActiveTag(tag)` | метод | Встановити або скинути тег-фільтр |
| `getPromptById(id)` | метод | Отримати промпт за ID |
